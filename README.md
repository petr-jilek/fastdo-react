## Development setup

### Requirements

- Node.js - v18.18.0

### Setup

- Clone repository
- Open in vs code
- Run task `Install packages` or by command `npm install`
- `npm install -g node-sass`
- You can run storybook with task `Run storybook` or by command `npm run storybook`
